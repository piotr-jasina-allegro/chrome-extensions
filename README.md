## Description
Chrome extension for JIRA:
- set Team field for new issues

## Installation
1. Clone repository.
2. Go to `chrome://extensions/` in a browser.
3. Click `Load unpacked` and choose a folder with plugin you want to install from this repository.

## How to update
1. Pull updates from the repository.
2. Go to `chrome://extensions/` in a browser.
3. Find extension you want to update and click refresh button :).
